DOM Calculator

Instructions

Presumably you know how to use a calculator but did you know you can operate this bad boi with the keyboard? Here's how:

  • - Press number keys to input numbers
  • - Press 'p' or '+' for the plus key
  • - Press '-' for the minus key
  • - Press 'x' or '*' for the multiply key
  • - Press '/' for the divide key
  • - Press 'Enter' or '=' for the equals key
  • - Press '.' for the point/decimal key
  • - Press 'c' or 'esc' for the clear key
